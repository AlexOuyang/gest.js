gest.js

gest.js is a webcam based gesture recognition library that helps developers make webpages more immersive.

USAGE

You will need to include the 'gest.js' script in your index <head>, using something like:

<script type="text/javascript" src="gest.js"></script>

Starting gest.js and handling gestures:

document.addEventListener('gest', function(gesture) {
    if (gesture.ready) {
        gest.start(); //start gesture detection
    } else {
        //handle gesture .direction .up .down .left .right here...
        document.getElementById('mydiv').innerHTML = gesture.direction;
    }
}, false);
gest.start();

To stop gesture detection, use:

gest.stop();

Toggle debug mode using:

gest.debug([boolean]);

Other available options: Enable/disable skin filtering

gest.options.skinFilter = [boolean]; //default is false

Enable/disable on screen messages

gest.options.messages = [boolean]; //default is true
